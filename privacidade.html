Política de Privacidade

Lembra-App — Assistente Pessoal de Lembretes e Agenda

← Voltar ao app

1. Informações Gerais

O Lembra-App é um aplicativo web pessoal de gerenciamento de lembretes, compromissos e integrações com serviços de agenda (Google Calendar e Microsoft Outlook). Esta política descreve como coletamos, usamos e protegemos suas informações.

2. Dados Coletados

O Lembra-App coleta apenas os dados estritamente necessários para seu funcionamento:

  • Dados de conta: nome e endereço de e-mail utilizados no cadastro
  • Lembretes e compromissos: títulos, datas, horários e categorias criados pelo usuário
  • Tokens de acesso OAuth: tokens temporários do Google e Microsoft para leitura e escrita na agenda, armazenados de forma segura no Supabase
  • Eventos de agenda: dados de eventos do Google Calendar e Outlook sincronizados pelo usuário

3. Uso das Informações

Os dados coletados são utilizados exclusivamente para:

  • Exibir e gerenciar lembretes e compromissos do usuário
  • Sincronizar eventos com Google Calendar e Microsoft Outlook
  • Enviar notificações de lembretes no dispositivo do usuário
  • Manter a sessão do usuário ativa entre dispositivos

Não vendemos, compartilhamos ou utilizamos seus dados para fins publicitários.

4. Integração com Google

O Lembra-App utiliza a API do Google Calendar com os seguintes escopos de acesso:

  • calendar.events — criar, editar e excluir eventos na agenda
  • calendar.readonly — ler eventos da agenda
  • userinfo.email — identificar o e-mail da conta conectada
  • gmail.readonly — verificar contagem de e-mails não lidos

O uso de dados obtidos via APIs do Google segue a Política de Dados do Usuário dos Serviços de API do Google, incluindo os requisitos de Uso Limitado.

5. Integração com Microsoft Outlook

O Lembra-App utiliza a API Microsoft Graph para leitura e escrita de eventos no Outlook Calendar e verificação de e-mails não lidos. Os tokens de acesso são armazenados localmente no dispositivo e nunca compartilhados com terceiros.

6. Armazenamento e Segurança

Os dados são armazenados de duas formas:

  • Localmente: no localStorage do navegador do usuário (lembretes, preferências)
  • Nuvem: no Supabase (plataforma segura com criptografia em trânsito e em repouso) para sincronização entre dispositivos

Tokens OAuth são armazenados de forma segura e nunca expostos publicamente.

7. Retenção de Dados

Os dados são mantidos enquanto o usuário utilizar o aplicativo. O usuário pode excluir todos os seus dados a qualquer momento acessando Configurações → Limpar dados dentro do app.

8. Direitos do Usuário

O usuário tem direito a:

  • Acessar todos os seus dados armazenados
  • Excluir sua conta e todos os dados associados
  • Revogar o acesso do app ao Google ou Outlook a qualquer momento em myaccount.google.com/permissions
